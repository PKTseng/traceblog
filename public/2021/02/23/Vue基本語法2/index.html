3. v-on 修飾符

在 JavaScript 裡面，都會監聽一些事件，例如 submit
但是我們都會用 event.preventDefault() 來阻擋事件冒泡，同理 v-on 也可以。
寫法也較為簡單:

  1. event.stopPropagation() 會寫成 .stop
  2. event.preventDefault() 會寫成 .prevent
  3. 只觸發一次 .once
  4. 只監聽特定按鍵，如: @keyup.enter 、 @keyup.按鍵碼
