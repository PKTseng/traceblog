元素選擇器

  1. 選擇要改變的元素 p

  3. 按下 button 的動作

DEMO

總結 id 、 className & 元素的差別:

  • id: 若有很多相同 id ，只會選取第一個
  • className: 會同時選取所有相同的 className
  • 元素: 會同時選取所有相同的元素

會用到 className 的原因是為了方便選更改多個樣式，可以用在相同的商品列表上，但如果選擇 id 的話就只會更改單一樣式而且是第一個，之後相同 id 的不會更改到
